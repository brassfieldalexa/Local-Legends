
LOCAL LEGENDS WEBSITE STARTER
=============================

Files included:
- index.html
- styles.css
- business_card_front.svg
- business_card_back.svg
- yard_sign.svg

HOW TO USE
----------
1. Open index.html in any web browser to preview the website.
2. Edit the parent contact info in index.html.
3. Upload index.html and styles.css to a free static host.
4. Print the SVG files as starter designs for your business cards and yard signs.

LOW-COST IDEA
-------------
This is built as a simple static website, so it can be hosted on free static hosting services.
